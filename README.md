# Express.js Practice

A Node.js/Express.js learning project exploring middleware concepts and basic server setup.

## Overview

This repository contains practice code for learning Express.js, focusing on middleware implementation, routing, and building basic RESTful APIs with Node.js.

## Topics Covered

- Express.js server setup
- Middleware implementation and chaining
- Route handling
- Request/response lifecycle

## Tech Stack

- **Node.js** — Runtime environment
- **Express.js** — Web framework

## Getting Started

```bash
npm install
node index.js
```
